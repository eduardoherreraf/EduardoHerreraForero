Manejo de Dataframes en Python

Introducci√≥n:Manejo de Dataframes en Python

Un DataFrame es una de las estructuras de datos m√°s potentes y utilizadas dentro de la biblioteca Pandas en Python, dise√±ada especialmente para el an√°lisis y la manipulaci√≥n de informaci√≥n tabular. Su principal caracter√≠stica es que organiza los datos en filas y columnas, de manera similar a una hoja de c√°lculo o una tabla en una base de datos.

Esta estructura permite trabajar con grandes vol√∫menes de informaci√≥n de forma eficiente, ofreciendo herramientas para filtrar, ordenar, combinar, agrupar y transformar datos con facilidad. Adem√°s, los DataFrames admiten distintos tipos de datos en cada columna (num√©ricos, de texto, fechas, booleanos, entre otros), lo que los hace extremadamente vers√°tiles.

Gracias a su flexibilidad, los DataFrames se han convertido en un componente esencial en tareas de an√°lisis de datos, ciencia de datos, machine learning y visualizaci√≥n, permitiendo a los desarrolladores y analistas extraer conocimiento valioso a partir de conjuntos de datos estructurados o no estructurados.

Definici√≥n: Dataframe

Un DataFrame en Pandas (Python) es una estructura de datos bidimensional que permite almacenar y manipular informaci√≥n en forma de tabla, similar a una hoja de c√°lculo de Excel o una tabla de base de datos.

Cada fila representa un registro u observaci√≥n, y cada columna representa una variable o campo de datos.

Caracter√≠sticas Principales de Dataframes

  • Estructura etiquetada: Las filas y columnas tienen etiquetas llamadas √≠ndices, lo que permite acceder f√°cilmente a los datos por nombre o posici√≥n.
  • Tipos de datos mixtos: Cada columna puede contener un tipo de dato distinto (n√∫meros, texto, fechas, valores booleanos, etc.).
  • Manipulaci√≥n sencilla: Permite filtrar, ordenar, agrupar, combinar y transformar datos de forma muy eficiente.
  • Integraci√≥n con otras herramientas: Es ampliamente usado en an√°lisis de datos, machine learning y visualizaci√≥n, y puede leer/escribir datos desde m√∫ltiples formatos como CSV, Excel, SQL, JSON, entre otros.

En resumen, un DataFrame es la herramienta principal de Pandas para manejar datos estructurados en Python, combinando la potencia de las listas y diccionarios con la facilidad de una tabla organizada.

Como se Usan los Dataframes

La diferencia entre loc e iloc en Pandas se resume as√≠:

loc iloc
Tipo de acceso Por etiquetas (nombres de filas o columnas) Por posici√≥n num√©rica (√≠ndices 0, 1, 2, ‚Ä¶)
Inclusi√≥n del rango final Incluye el l√≠mite final Excluye el l√≠mite final
Ejemplo df.loc['a', 'Edad'] ‚Üí usa nombres df.iloc[0, 1] ‚Üí usa posiciones
Uso recomendado Cuando se conocen las etiquetas Cuando se conocen las posiciones

Razones del porqu√© usar loc y no iloc:

  • Mayor legibilidad: loc hace que el c√≥digo sea m√°s comprensible al usar nombres en lugar de n√∫meros.
  • Menor riesgo de errores: Las etiquetas no cambian si se reordenan las filas o columnas, lo que evita confusiones.
  • M√°s intuitivo: Trabajar con nombres de columnas es m√°s natural en an√°lisis de datos.
  • Acceso combinado: Permite filtrar y seleccionar datos por etiquetas y condiciones en una sola l√≠nea.

Importaci√≥n de Archivos CSV

La capacidad de importar y exportar datos es fundamental en el an√°lisis de datos, ya que la mayor√≠a de los datos que utilizamos provienen de fuentes externas, como archivos CSV, Excel, bases de datos, o APIs. En Python, la librer√≠a Pandas simplifica enormemente estas tareas.

Estructura B√°sica

Un archivo CSV tiene una estructura muy simple:

                      
                        Nombre,Edad,Pa√≠s
                        Ana,25,Colombia
                        Luis,30,M√©xico
                        Mar√≠a,28,Argentina
  • La primera l√≠nea suele contener los nombres de las columnas (encabezados).
  • Cada fila siguiente contiene los valores de cada registro.
  • Los campos se separan por comas, aunque en algunos pa√≠ses o configuraciones se usa punto y coma (;).

Caracter√≠sticas Principales

  • Es un archivo de texto plano, por lo que puede abrirse con cualquier editor (como Bloc de notas).
  • Es ligero y f√°cil de compartir.
  • Puede ser le√≠do por hojas de c√°lculo (Excel, Google Sheets) y por lenguajes de programaci√≥n como Python.
  • No guarda formato visual (colores, f√≥rmulas o estilos), solo datos estructurados.

Ventajas de CSV

Importaci√≥n de Archivos Excel

La importaci√≥n de archivos Excel en pandas con Python es una tarea fundamental en el an√°lisis de datos, ya que permite leer informaci√≥n almacenada en hojas de c√°lculo y convertirla en estructuras de datos manipulables, como los DataFrames. A continuaci√≥n se explica paso a paso c√≥mo se realiza este proceso.

Requisitos Previos para Importar Documentos Excel

Antes de importar archivos Excel, es necesario tener instalada la biblioteca pandas y el motor de lectura de Excel, como openpyxl o xlrd.

En la terminal o consola de Python se deben ejecutar los siguientes comandos:

                    
                     pip install pandas openpyxl

Nota: openpyxl se utiliza para leer y escribir archivos Excel con extensi√≥n .xlsx. En versiones actuales de pandas, xlrd ya no soporta este formato.

Cargar un Archivo Excel

'datos.xlsx' es el nombre del archivo que se encuentra en la misma carpeta del script.

Si el archivo est√° en otra ubicaci√≥n, se debe incluir la ruta completa:

Un archivo Excel puede tener varias hojas.

Para leer una hoja espec√≠fica se usa el par√°metro sheet_name:

Tambi√©n se puede leer varias hojas al mismo tiempo (retorna un diccionario de DataFrames):

Par√°metros √ötiles en read_excel()

Par√°metro Descripci√≥n Ejemplo

nrows Limita el n√∫mero de filas le√≠das nrows=10
